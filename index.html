Stremebase

A low level embeddable multi-model storage engine tightly coupled with a high level java 8 streams api compliant interface.

import com.stremebase.base.DB;
import com.stremebase.map.OneMap;

DB.startDB(true);

OneMap map = new OneMap("map");
long key = map.getLargestKey();
for (long i=0; i<100000; i++) map.put(++key, i%100);
map.commit();
map.query(47, 58).forEach(key->printf("%d->%d", key, map.get(key));

Getting Started

1. Get yourself Java 8 JDK, if you don't already have it.

2. Download stremebase.jar

3. Run the tutorials; start with this

4. Study the javadocs
